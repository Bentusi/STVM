## 🔧 PID 原理

**公式**：`output = Kp*error + Ki*∫error + Kd*Δerror`

- **Kp** - 比例：响应速度
- **Ki** - 积分：消除稳态误差  
- **Kd** - 微分：减少超调

## 🎛️ 参数调整

编辑 `pid_test.st` 修改参数：

```st
(* 保守 - 稳定 *)
Kp := 0.3;
Ki := 0.05;
Kd := 0.02;

(* 激进 - 快速 *)
Kp := 1.0;
Ki := 0.3;
Kd := 0.1;

(* 纯P - 有误差 *)
Kp := 0.5;
Ki := 0.0;
Kd := 0.0;
```

## 🐛 常见问题

| 问题 | 原因 | 解决 |
|------|------|------|
| 振荡 | Kp过大 | 减小Kp |
| 太慢 | Kp过小 | 增大Kp |
| 有误差 | Ki太小 | 增大Ki |
| 超调 | Kp大/Kd小 | 减Kp增Kd |
