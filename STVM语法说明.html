STVM 语法说明

概述

STVM (Structured Text Virtual Machine) 是一个基于 IEC 61131-3 标准的结构化文本 (ST) 语言虚拟机。本文档详细说明了 STVM 支持的语法特性和使用注意事项。

一、程序结构

1.1 基本程序结构

PROGRAM ProgramName
    (* 可选：导入声明 *)
    (* 可选：变量声明 *)
    (* 可选：函数定义 *)
    (* 语句 *)
END_PROGRAM

或使用显式的 BEGIN 块：

PROGRAM ProgramName
    (* 可选：导入声明 *)
    (* 可选：变量声明 *)
    (* 可选：函数定义 *)
BEGIN
    (* 语句 *)
END_PROGRAM

1.2 函数结构

有返回值的函数：

FUNCTION FunctionName : ReturnType
VAR_INPUT
    param1 : Type1;
    param2 : Type2;
END_VAR
VAR_LOCAL
    local1 : Type3;
END_VAR
    (* 语句 *)
END_FUNCTION

无返回值的函数（PROCEDURE 风格）：

FUNCTION FunctionName
VAR_INPUT
    param1 : Type1;
END_VAR
    (* 语句 *)
END_FUNCTION

⚠️ 注意： - 有返回值的函数通过函数名赋值返回：FunctionName := result; - 无返回值的函数类似过程（PROCEDURE），只执行操作不返回值 - 函数调用可以作为独立语句（无返回值或忽略返回值）

二、数据类型

2.1 基本数据类型

类型 说明 示例
BOOL 布尔型 TRUE, FALSE
INT 整数型 (64位) 42, -10, 0
REAL 实数型 (双精度) 3.14, 2.5e-3
STRING 字符串型 'Hello', "World"

2.2 质量化数据类型

质量化类型包含 .VAL (值) 和 .QUALITY (质量) 两个成员：

类型 说明 成员访问
QBOOL 质量化布尔 var.VAL, var.QUALITY
QINT 质量化整数 var.VAL, var.QUALITY
QREAL 质量化实数 var.VAL, var.QUALITY
QSTRING 质量化字符串 var.VAL, var.QUALITY

示例：

VAR
    temp : QREAL;
END_VAR
    temp.VAL := 25.5;
    temp.QUALITY := 0;

2.3 数组类型

支持一维和多维数组：

(* 方式 1: 指定范围 *)
VAR
    arr1 : ARRAY[0..9] OF INT;     (* 10个元素，索引 0-9 *)
END_VAR

(* 方式 2: 指定大小 *)
VAR
    arr2 : ARRAY[10] OF REAL;      (* 10个元素，索引 0-9 *)
END_VAR

数组访问：

arr1[0] := 100;
x := arr1[5];

三、变量声明

3.1 变量声明的位置限制

⚠️ 重要限制：变量只能在以下位置声明：

  1. 程序开头（PROGRAM 之后、语句之前）
  2. 函数开头（FUNCTION 之后、BEGIN 或语句之前）

❌ 不能在语句中间声明变量

正确示例：

PROGRAM Example
VAR
    x : INT;        (* ✓ 在程序开头 *)
END_VAR
    x := 10;
END_PROGRAM

错误示例：

PROGRAM Example
    x := 10;
    VAR             (* ✗ 不能在语句后声明 *)
        y : INT;
    END_VAR
END_PROGRAM

3.2 变量块类型

VAR - 全局变量

VAR
    global_var : INT := 100;
END_VAR
  • 在 PROGRAM 中：全局变量
  • 在 FUNCTION 中：静态全局变量（函数内也可定义全局变量）

VAR_INPUT - 函数输入参数

FUNCTION Add : INT
VAR_INPUT
    x : INT;
    y : INT;
END_VAR
    Add := x + y;
END_FUNCTION

VAR_OUTPUT - 函数输出参数

VAR_OUTPUT
    result : INT;
END_VAR

VAR_LOCAL - 函数局部变量

VAR_LOCAL
    temp : INT;
    counter : INT := 0;
END_VAR

VAR_EXTERNAL - 外部 I/O 映射

用于映射物理 I/O 地址：

VAR_EXTERNAL
    (* 方式 1: 使用字符串地址 *)
    button AT "GPIO_27" : BOOL;
    
    (* 方式 2: 使用直接地址 *)
    led AT %QX0.0 : BOOL;
    sensor AT %IX1.2 : INT;
END_VAR

直接地址格式：%<位置><尺寸><地址> - 位置：I (输入), Q (输出), M (内存) - 尺寸：X (位), B (字节), W (字), D (双字), L (长字) - 地址：如 0.0, 1.2

3.3 变量初始化

支持在声明时初始化：

VAR
    x : INT := 42;                  (* 整数初始化 *)
    pi : REAL := 3.14159;          (* 实数初始化 *)
    flag : BOOL := TRUE;           (* 布尔初始化 *)
    msg : STRING := 'Hello';       (* 字符串初始化 *)
END_VAR

3.4 空变量块

支持空的变量声明块：

VAR
END_VAR

VAR_INPUT
END_VAR

VAR_LOCAL
END_VAR

四、运算符

4.1 算术运算符

运算符 说明 示例
+ 加法 a + b
- 减法 / 负号 a - b, -x
* 乘法 a * b
/ 除法 a / b
MOD 取模 a MOD b

4.2 比较运算符

运算符 说明 示例
= 等于 a = b
<> 不等于 a <> b
< 小于 a < b
<= 小于等于 a <= b
> 大于 a > b
>= 大于等于 a >= b

4.3 逻辑运算符

运算符 说明 示例
AND 逻辑与 a AND b
OR 逻辑或 a OR b
XOR 逻辑异或 a XOR b
NOT 逻辑非 NOT a

4.4 位运算符

运算符 说明 示例
SHL 左移 x SHL 2
SHR 右移 x SHR 1

4.5 运算符优先级

从高到低（符合 IEC 61131-3 标准）：

  1. 括号 ( )
  2. 一元运算符：NOT, - (负号)
  3. 乘除：*, /, MOD
  4. 加减和移位：+, -, SHL, SHR
  5. 比较：<, <=, >, >=
  6. 相等：=, <>
  7. 逻辑与：AND
  8. 逻辑异或：XOR
  9. 逻辑或：OR

五、语句

5.1 赋值语句

使用 := 进行赋值：

(* 简单赋值 *)
x := 10;

(* 数组赋值 *)
arr[0] := 100;

(* 质量化成员赋值 *)
temp.VAL := 25.5;
temp.QUALITY := 0;

⚠️ 注意：赋值使用 :=，判断相等使用 =

5.2 条件语句 (IF)

(* 基本形式 *)
IF condition THEN
    statements;
END_IF;

(* 带 ELSE *)
IF condition THEN
    statements1;
ELSE
    statements2;
END_IF;

(* 带 ELSIF *)
IF condition1 THEN
    statements1;
ELSIF condition2 THEN
    statements2;
ELSIF condition3 THEN
    statements3;
ELSE
    statements4;
END_IF;

5.3 选择语句 (CASE)

CASE expression OF
    1: statement1;
    2: statement2;
    3, 4, 5: statement3;    (* 多标签 *)
ELSE
    default_statement;
END_CASE;

示例：

CASE x OF
    1: result := 'One';
    2: result := 'Two';
    3: result := 'Three';
ELSE
    result := 'Other';
END_CASE;

5.4 循环语句

WHILE 循环

WHILE condition DO
    statements;
END_WHILE;

示例：

counter := 0;
WHILE counter < 10 DO
    counter := counter + 1;
END_WHILE;

FOR 循环

(* 基本形式，步长为 1 *)
FOR i := 1 TO 10 DO
    statements;
END_FOR;

(* 指定步长 *)
FOR i := 0 TO 100 BY 10 DO
    statements;
END_FOR;

示例：

sum := 0;
FOR i := 1 TO 10 DO
    sum := sum + i;
END_FOR;

REPEAT 循环

REPEAT
    statements;
UNTIL condition
END_REPEAT;

REPEAT 循环

REPEAT
    statements;
UNTIL condition
END_REPEAT;

示例：

counter := 1;
sum := 0;
REPEAT
    sum := sum + counter;
    counter := counter + 1;
UNTIL counter > 10
END_REPEAT;

⚠️ 注意： - REPEAT 循环至少执行一次（先执行循环体，再检查条件） - 当 UNTIL 后的条件为真时退出循环 - 与 WHILE 循环的区别：WHILE 是条件为真时执行，REPEAT 是条件为真时退出

循环对比

循环类型 条件检查时机 最少执行次数 退出条件
WHILE 循环开始前 0次 条件为假
FOR 循环开始前 0次 超出范围
REPEAT 循环结束后 1次 条件为真

5.5 函数调用

在表达式中使用（获取返回值）：

(* 无参数函数 *)
result := GetValue();

(* 带参数函数 *)
sum := Add(5, 10);
result := Calculate(x, y, z);

作为独立语句（无返回值或忽略返回值）：

(* 调用无返回值函数 - PROCEDURE 风格 *)
PrintMessage('Hello');
DoSomething();

(* 调用有返回值函数但忽略返回值 *)
Calculate(x, y);  (* 返回值被忽略 *)

⚠️ 注意： - 有返回值的函数可以在表达式中使用，也可以作为语句调用（忽略返回值） - 无返回值的函数只能作为语句调用，不能在表达式中使用

5.6 返回语句

(* 无返回值 *)
RETURN;

(* 有返回值 *)
RETURN expression;

(* 函数返回值赋值 *)
FunctionName := result_value;

5.7 打印语句

(* 单参数 *)
PRINT('Hello World');

(* 多参数（格式化） *)
PRINT('Result: %d\n', result);
PRINT('x=%d, y=%d\n', x, y);

格式化占位符： - %d - 整数 - %f - 实数 - %s - 字符串 - %b - 布尔

IMPORT Abs FROM 'examples/mathlib.stbc';

IMPORT Abs AS MyAbs FROM 'examples/mathlib.stbc';

⚠️ 注意： - 导入语句必须在 PROGRAM 之后、变量声明之前 - 当前仅支持从编译库 (.stbc) 导入

九、标识符规则

9.1 命名规则

  • 必须以字母或下划线开头：a-z, A-Z, _
  • 后续可包含字母、数字、下划线：a-z, A-Z, 0-9, _
  • 不区分大小写（但建议保持一致）

合法标识符：

myVariable
_temp
counter123
MyFunction
GPIO_27

非法标识符：

123abc      (* 不能以数字开头 *)
my-var      (* 不能包含连字符 *)
my var      (* 不能包含空格 *)

9.2 保留关键字

以下关键字不能用作标识符：

程序结构：PROGRAM, END_PROGRAM, FUNCTION, END_FUNCTION, BEGIN

变量声明：VAR, VAR_INPUT, VAR_OUTPUT, VAR_LOCAL, VAR_EXTERNAL, END_VAR, AT

控制流：IF, THEN, ELSIF, ELSE, END_IF, CASE, OF, END_CASE, FOR, TO, BY, DO, END_FOR, WHILE, END_WHILE, REPEAT, UNTIL, END_REPEAT, RETURN, EXIT

数据类型：BOOL, INT, REAL, STRING, ARRAY, QBOOL, QINT, QREAL, QSTRING

运算符：AND, OR, XOR, NOT, MOD, SHL, SHR

其他：TRUE, FALSE, PRINT, IMPORT, FROM, AS, VAL, QUALITY

十、完整示例

10.1 Hello World

PROGRAM HelloWorld
VAR
    message : STRING := 'Hello, STVM!';
    counter : INT := 0;
END_VAR

    counter := 42;
    PRINT(message);
    PRINT('Counter: %d\n', counter);
END_PROGRAM

10.2 函数示例

PROGRAM Functions
VAR
    a : INT := 5;
    b : INT := 3;
    result : INT;
END_VAR

FUNCTION Add : INT
VAR_INPUT
    x : INT;
    y : INT;
END_VAR
    Add := x + y;
END_FUNCTION

FUNCTION Factorial : INT
VAR_INPUT
    n : INT;
END_VAR
VAR_LOCAL
    i : INT;
    temp : INT;
END_VAR
    temp := 1;
    FOR i := 1 TO n DO
        temp := temp * i;
    END_FOR;
    Factorial := temp;
END_FUNCTION

    result := Add(a, b);
    PRINT('Sum: %d\n', result);
    
    result := Factorial(5);
    PRINT('Factorial: %d\n', result);
END_PROGRAM

10.3 无返回值函数示例 (VOID/PROCEDURE 风格)

PROGRAM VoidFunctionDemo
VAR
    result : INT := 0;
END_VAR

(* 无返回值函数 - 简单打印 *)
FUNCTION PrintHeader
    PRINT('===================\n');
    PRINT('  STVM Demo\n');
    PRINT('===================\n');
END_FUNCTION

(* 无返回值函数 - 带参数 *)
FUNCTION PrintValue
VAR_INPUT
    label : STRING;
    value : INT;
END_VAR
    PRINT('%s: %d\n', label, value);
END_FUNCTION

(* 有返回值函数 - 对比 *)
FUNCTION Multiply : INT
VAR_INPUT
    x : INT;
    y : INT;
END_VAR
    Multiply := x * y;
END_FUNCTION

BEGIN
    (* 调用无返回值函数 - 只能作为语句 *)
    PrintHeader();
    PrintValue('Test Value', 42);
    
    (* 调用有返回值函数 - 获取返回值 *)
    result := Multiply(6, 7);
    PRINT('Result: %d\n', result);
    
    (* 调用有返回值函数 - 忽略返回值 *)
    Multiply(3, 4);
    
    PrintHeader();
END_PROGRAM

10.4 I/O 映射示例

PROGRAM LEDBlink

VAR_EXTERNAL
    button AT %IX0.0 : BOOL;    (* GPIO输入 *)
    led AT %QX0.0 : BOOL;       (* GPIO输出 *)
END_VAR

VAR
    blink_timer : INT := 0;
    led_state : BOOL := FALSE;
END_VAR

    IF button THEN
        blink_timer := blink_timer + 1;
        
        IF blink_timer >= 500 THEN
            led_state := NOT led_state;
            led := led_state;
            blink_timer := 0;
        END_IF;
    ELSE
        led := FALSE;
        blink_timer := 0;
    END_IF;
END_PROGRAM

10.5 质量化数据示例

PROGRAM QualityTest
VAR
    temperature : QREAL;
    sensor_ok : BOOL;
END_VAR

    temperature.VAL := 25.5;
    temperature.QUALITY := 192;  (* Good quality *)
    
    sensor_ok := (temperature.QUALITY = 192);
    
    IF sensor_ok THEN
        PRINT('Temp: %f C\n', temperature.VAL);
    ELSE
        PRINT('Sensor error!\n');
    END_IF;
END_PROGRAM

十一、循环语句详解

11.1 WHILE 循环

语法：先判断条件，条件为真时执行循环体

WHILE condition DO
    statements;
END_WHILE;

特点： - 条件在循环开始前检查 - 如果初始条件为假，循环体一次都不执行 - 适合不确定循环次数的场景

示例：

counter := 0;
WHILE counter < 10 DO
    counter := counter + 1;
END_WHILE;
(* 如果 counter 初始值 >= 10，循环体不会执行 *)

11.2 FOR 循环

语法：指定范围的计数循环

FOR variable := start TO end DO
    statements;
END_FOR;

FOR variable := start TO end BY step DO
    statements;
END_FOR;

特点： - 循环次数确定 - 循环变量自动递增 - 步长可选，默认为 1 - 条件在循环开始前检查

示例：

(* 1 到 10 *)
FOR i := 1 TO 10 DO
    sum := sum + i;
END_FOR;

(* 0 到 100，步长 10 *)
FOR i := 0 TO 100 BY 10 DO
    PRINT('%d\n', i);
END_FOR;

11.3 REPEAT 循环

语法：至少执行一次，然后检查退出条件

REPEAT
    statements;
UNTIL condition
END_REPEAT;

特点： - 循环体至少执行一次 - 条件在循环结束后检查 - 当条件为真时退出循环 - 适合需要先执行后判断的场景

示例：

(* 输入验证 - 至少询问一次 *)
REPEAT
    PRINT('Enter positive number: ');
    (* ... 读取输入 ... *)
UNTIL input > 0
END_REPEAT;

(* 累加到大于 100 *)
sum := 0;
counter := 1;
REPEAT
    sum := sum + counter;
    counter := counter + 1;
UNTIL sum > 100
END_REPEAT;

11.4 循环对比总结

循环类型 条件检查时机 最少执行次数 退出条件 适用场景
WHILE 循环开始前 0次 条件为假时退出 不确定次数
FOR 循环开始前 0次 超出范围时退出 确定次数
REPEAT 循环结束后 1次 条件为真时退出 至少一次

十二、常见错误和注意事项

12.1 变量声明位置错误

❌ 错误：

PROGRAM Test
    x := 10;
    VAR y : INT; END_VAR    (* 错误：不能在语句后声明 *)
END_PROGRAM

✓ 正确： ### 12.1 变量声明位置错误

❌ 错误：

PROGRAM Test
    x := 10;
    VAR y : INT; END_VAR    (* 错误：不能在语句后声明 *)
END_PROGRAM

✓ 正确：

PROGRAM Test
VAR
    y : INT;
END_VAR
    x := 10;
END_PROGRAM

12.2 赋值符号混淆

❌ 错误：

x = 10;          (* 错误：= 是比较运算符 *)

✓ 正确：

x := 10;         (* 正确：:= 是赋值运算符 *)
IF x = 10 THEN   (* 正确：= 用于比较 *)

12.3 字符串引号转义

❌ 错误：

msg := 'It's wrong';     (* 错误：未转义单引号 *)

✓ 正确：

msg := 'It''s correct';  (* 正确：单引号转义 *)
msg := "It's also OK";   (* 或使用双引号 *)

12.4 数组索引从 0 开始

VAR
    arr : ARRAY[0..9] OF INT;    (* 10个元素 *)
END_VAR
    arr[0] := 100;   (* 第一个元素 *)
    arr[9] := 200;   (* 最后一个元素 *)

12.5 REPEAT 循环条件理解

❌ 容易混淆：

counter := 0;
REPEAT
    counter := counter + 1;
UNTIL counter < 5        (* 错误理解：以为是继续条件 *)
END_REPEAT;
(* 实际：counter = 1 时就退出了！*)

✓ 正确理解：

counter := 0;
REPEAT
    counter := counter + 1;
UNTIL counter >= 5       (* 正确：条件为真时退出 *)
END_REPEAT;
(* counter 达到 5 时退出，循环了 5 次 *)

⚠️ 关键区别： - WHILE condition DO - condition 为真时继续 - REPEAT ... UNTIL condition - condition 为真时退出

12.6 函数返回值赋值方式

11.2 赋值符号混淆

❌ 错误：

x = 10;          (* 错误：= 是比较运算符 *)

✓ 正确：

x := 10;         (* 正确：:= 是赋值运算符 *)
IF x = 10 THEN   (* 正确：= 用于比较 *)

11.3 字符串引号转义

❌ 错误：

msg := 'It's wrong';     (* 错误：未转义单引号 *)

✓ 正确：

msg := 'It''s correct';  (* 正确：单引号转义 *)
msg := "It's also OK";   (* 或使用双引号 *)

12.6 函数返回值赋值方式

FUNCTION GetValue : INT
    GetValue := 42;    (* 通过函数名赋值返回 *)
END_FUNCTION

12.7 关键字大小写

所有关键字不区分大小写：

PROGRAM Test        (* ✓ *)
Program Test        (* ✓ *)

VAR x : INT;        (* ✓ *)
var x : int;        (* ✓ *)

但为了代码可读性，建议保持一致的风格。
